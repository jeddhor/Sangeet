About Sangeet

Sangeet is a vibrant Navidrome web client focused on smooth playback, fast browsing, and a joyful music-first experience.

  • Three-pane desktop layout with responsive mobile views.
  • Library navigation for tracks, genres, favorites, and playlists.
  • Now Playing with album art, lyrics, and transport controls.
  • Theme support with colorful personalities.

Built with Node.js + Express and powered by the Subsonic-compatible Navidrome API.
